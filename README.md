# Women's Communication Lab LP

GitHub Pagesで公開できる完成版LPです。

## 公開方法
1. ZIPを解凍
2. `index.html` と `assets` フォルダをGitHubリポジトリのルートへアップロード
3. GitHubの `Settings` → `Pages`
4. `Deploy from a branch` を選び、Branchを `main`、Folderを `/root` に設定

## 必ず差し替える箇所
`index.html` 内の `@YOUR_LINE_ID` を、実際のLINE公式アカウントURLに置き換えてください。
